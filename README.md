# oc-p08-Knime_RGPD

Traitement SIRH multi-sources, anonymisation RGPD, calcul index légal + 5 indicateurs clés (salaires ETP, promotions, répartition CSP). Visualisations automatisées. Démontre maîtrise ETL, conformité réglementaire et data governance
